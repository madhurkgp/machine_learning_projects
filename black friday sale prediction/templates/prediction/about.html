How It Works

  1. Data Collection
    Customer demographic and transaction data is collected through the input form.
  2. Data Preprocessing
    Raw data is cleaned, encoded, and scaled to match the training data format.
  3. Feature Engineering
    Categorical variables are encoded and numerical features are standardized.
  4. Model Prediction
    The preprocessed data is fed to the trained Random Forest model.
  5. Result Display
    The predicted purchase amount is displayed with confidence metrics.
