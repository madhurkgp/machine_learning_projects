About Sonic Log Predictor

Advanced Machine Learning for Well Log Analysis

Project Overview

The Sonic Log Predictor is an advanced machine learning system designed to predict DTC (Compressional Travel-time) and DTS (Shear Travel-time) values from well log data using state-of-the-art XGBoost algorithms enhanced with Wavelet Transformation.

This project demonstrates the application of signal processing techniques and machine learning to solve real-world problems in petroleum engineering and geoscience. By leveraging wavelet transformation, we achieve superior accuracy compared to traditional methods.

High Accuracy

Achieves R² scores of 0.982 for DTC and 0.999 for DTS predictions

Wavelet Transformation

Advanced signal processing for enhanced feature extraction

Real-time Processing

Fast predictions suitable for field applications

Technical Implementation

Machine Learning Pipeline
  1. Data Preprocessing: Handle missing values, outliers, and negative values
  2. Feature Engineering: Log transformation for resistivity logs
  3. Wavelet Transformation: Apply db4 wavelet to CNC log
  4. Model Training: XGBoost with optimized hyperparameters
  5. Prediction: Real-time inference with confidence scoring
Input Features
  • CAL: Caliper (Inch)
  • CNC: Neutron Log (decimal)
  • GR: Gamma Ray (API)
  • HRD: Deep Resistivity (Ohm/m)
  • HRM: Medium Resistivity (Ohm/m)
  • PE: Photo-electric Factor (Barn)
  • ZDEN: Density (g/m³)
